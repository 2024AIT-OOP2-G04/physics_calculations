# 村中＠GPTによる​物理実験T12サポートサイト​

## アプリ名
　村中＠GPTによる​物理実験T12サポートサイト​


## アプリの概要

　このアプリは、物理実験T12のグラフ出力と、値の計算を行ってくれるアプリ
  
  物理実験T12は、グラフを5つも作成する必要があり、数ある実験の中でも時間がかかる
 
  実際、愛工大の2024年度のkkだと、2025年10月1日時点で30人くらいがT12のレポートを出し遅れてたり、出していなかったりする
  
  そんなときにこのアプリを使えば、一番面倒なグラフと計算結果の出力をしてくれる

　面倒なグラフ描写や計算はこのアプリにお任せ！！

##　動作条件
  このリポジトリをクローンし、必要なライブラリをインストールする
 
## 使い方

- app.jsを実行し、出力されたサーバーのurlより「村中＠GPTによる​物理実験T12サポートサイト」を開く
- グラフ描写用の入力フォームに値を入力
- 「グラフを出力」ボタンを押してグラフを出力
- グラフより計算用のデータを読み取り、入力フォームに値を入力
- 「計算を実行」ボタンを押して計算結果を出力

## 必要な機能
- 入力フォームに値を入力できる
- ボタンを押したらグラフが出力される
- ボタンを押したら計算結果が出力される

## 　各データの形式

  ### ip_vg特性曲線用のinputデータ
  平均値のみ 要素数9の配列

  ```json
  "input_ip_vg":{
    "vp_230":[0,0.10,0.5,1.48,3.30,6.23,10.05,15.30,21.75],
    "vp_250":[0.05,0.35,1.10,2.63,4.98,8.38,12.83,18.45,25.55],
    "vp_270":[0.25,0.83,2.03,4.08,6.95,10.88,15.83,21.98,29.53]
  }
  ```

  ### gm計算用の値 json、辞書型データ
  
  ```json
  "calculation_gm":{
    // 動作点のip
    "ip_default":12.90,
    //vg=-6のときのip
    "vg_minus6_ip":17.70,
    // ip=0のときのvg
    // この値は絶対値にして計算したい
    "ip_0_vg":-13.20
  }  
  ```

### ip_vp特性曲線用のinputデータ
 
  -6,-8,-10Vのデータを作成する
  要素数3の配列

  ```json
  "input_ip_vp":{
    "vg_minus6":[
      15.30,18.45,21.98
    ],
    "vg_minus8":[
      10.5,12.83,15.83
    ],
    "vg_minus10":[
      6.23,8.38,10.88
    ]
  }  
  ```

  ### gamma_p計算用の値 json、辞書型データ
  
  ```json
  "calculation_gamma_p":{
    //vp=180のときのip(vg=-8V)
    "vp_180_ip":3.00,
    //vp=300のときのip(vg=-8V)
    "vp_300_ip":20.3,
  }  
  ```

  ### vp_vg特性曲線用のinputデータ
  
  ip_defaultは接線を引いたところのip
  要素数3の配列

  ```json
  "input_vp_vg":{
    // 動作点のip
    "ip_default":12.90,
    "vg_minus5":[
      15.30,18.45,21.98
    ],
    "vg_default":[
      10.5,12.83,15.83
    ],
    "vg_plus5":[
      6.23,8.38,10.88
    ]
  }  
  ```

  ### mu計算用のデータ
  ```json
 "calculation_mu":{
  "vp_180_vg":-3.90,
  "vp_300_vg":-10.80
}
  ```
